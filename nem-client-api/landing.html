How NCC works?

NCC provides an access to your assets and NEMs like a traditional wallet does. You may

  • Have multiple wallets
  • Define multiple accounts to be included in a wallet

NCC requires access to an NIS server in order to operate. Standard is to have a local server active (is installed together with the NCC)

You may also configure an access to a remote NIS.

What is *NIS?

This component is responsible for keeping the NEM cloud alive.

The more NIS the better the security.

NIS is the access point into the NEM cloud.


*NIS stands for NEM Infrastructure Server
